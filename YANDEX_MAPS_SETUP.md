# Настройка Яндекс.Карт для главной страницы

## Получение API-ключа

1. Перейдите на [https://developer.tech.yandex.ru/](https://developer.tech.yandex.ru/)
2. Войдите или зарегистрируйтесь
3. Перейдите в раздел "JavaScript API и HTTP Геокодер"
4. Нажмите "Получить ключ"
5. Заполните форму и получите ключ

## Установка API-ключа

Откройте файл `resources/views/index.blade.php` и найдите строку:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_API_KEY&lang=ru_RU" type="text/javascript"></script>
```

Замените `YOUR_API_KEY` на ваш полученный ключ:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ваш-ключ-здесь&lang=ru_RU" type="text/javascript"></script>
```

## Настройка координат офиса

В файле `resources/views/index.blade.php` найдите функцию `init()` и измените координаты:

```javascript
center: [55.751574, 37.573856], // Замените на координаты вашего офиса
```

### Как узнать координаты:

1. Откройте [Яндекс.Карты](https://yandex.ru/maps/)
2. Найдите нужный адрес
3. Кликните правой кнопкой мыши на точке
4. Выберите "Что здесь?"
5. Скопируйте координаты

## Настройка адреса в метке

Измените текст в balloonContent:

```javascript
balloonContent: '<strong>Golden Models</strong><br>Ваш адрес<br><a href="tel:+79991234567">Ваш телефон</a>'
```

## Тестирование

После настройки откройте главную страницу и прокрутите до раздела "Контакты". Карта должна отобразиться с вашей меткой.

## Важно

- Без API-ключа карта работать не будет
- Бесплатный лимит: 25000 запросов в сутки
- Для продакшена рекомендуется добавить домен в белый список в личном кабинете Яндекс
